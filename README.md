# Algos
Practice Site for Algorithms and Cryptography

## Setup

This project uses [uv](https://docs.astral.sh/uv/) as the package manager.

### Prerequisites
- Python 3.12+
- uv package manager

### Installing uv
```bash
pip install uv
```

### Running the project
```bash
# Run the main application
uv run main.py

# Sync dependencies
uv sync

# Add new dependencies
uv add <package-name>
```
